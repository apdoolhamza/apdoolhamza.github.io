## 🚀 Getting Started

### Installation

- Step 1: Open Terminal(on Mac) or Git Bash(on Window)

- Step 2: Navigate to the directory where you’d like your cloned repository to appear using 'cd'(change directory) command.
Example:

   ```bash
   cd '/C/Users/Your System Username/myPortfolio'
   ```
   
- Step 3: Clone this repository:
   copy and paste this link

   ```bash
   git clone https://github.com/apdoolhamza/apdoolhamza.github.io.git
   ```

- Step 4: Open `index.html` in your web browser.
